# 截图说明

运行 GUI 后截取以下截图，放在此目录下：

| 文件名 | 对应页面 | 说明 |
|--------|---------|------|
| `aide-management.png` | AIDE 管理 | IDE 检测列表 + CLI/App 标签 + 操作按钮 |
| `session-drawer.png` | AIDE 管理 - 会话抽屉 | 点击「会话」按钮后右侧滑出的会话列表 |
| `llm-config.png` | LLM 配置 | Provider 列表 + 智能添加 + 验证 Key |
| `skills-config.png` | Skills 配置 | 本地预置 tab + 已安装技能列表 |

截图工具：
- macOS: `Cmd + Shift + 4` 区域截图
- Windows: `Win + Shift + S` 区域截图
